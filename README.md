# darkosre-additionals

## Introduction
This repository contains some additional scripts taken from AeolusUX/ArkOS-R3XS that I find useful.

## Instructions
1. download as zip and extract.
2. Copy Additionals folder into your ROMS partition's tools folder of the SD card.
3. On your R36S, under Options > Tools, you should see the Additionals folder you copied.
4. Run any of the script inside the folder as you like. 

## Other details
There is an embedded script (/usr/local/bin/r36_config.sh) that runs during boot so the script like changing LED is only temporary and will reset at boot.

It is perhaps useful at night where you want to change LED to red.
